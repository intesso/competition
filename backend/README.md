# competition backend

## install

prerequisites: 

- install node.js LTS: https://nodejs.org/en/download/

open a terminal at this directory and run:

```sh
npm install
```


## start application

```sh
npm start
```

- https://nodejs.org/en/download/

- backend at http://localhost:8081
